# Converted Tools Workspace

Start here: [../../README.md](../../README.md)

This folder contains standalone SPA scaffolds for each Wizardawn tool conversion.

## Structure

- `osr-support-files/shared/css/global.css`
- `osr-support-files/shared/js/core.js`
- `osr-support-files/shared/js/data.js`
- `osr-support-files/shared/js/map-utils.js`
- `osr-support-files/assets/*`
- `osr-support-files/db-json-export/*`
- `osr-<tool-id>.html`
- `osr-<tool-id>/app.js`
- `osr-<tool-id>/styles.css`

## Notes

- Tools are scaffolded only; generation logic still needs to be ported from PHP.
- Tools inherit tedt.org global styling via `/tools/lib/tedt-org-tools.css`.
- Export stubs (JSON/Markdown/HTML) are included in each tool app.
