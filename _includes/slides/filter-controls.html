{% comment %} Slide topic filter controls. Builds a unique topic list from site.slides front matter `topics` arrays. Client-side toggling: shows cards containing ANY selected topic when one or more selected; if none selected, shows all. Accessibility: Buttons are toggle buttons with aria-pressed state. {% endcomment %}
{%- assign all_topics = '' | split: '' -%} {%- for deck in site.slides -%} {%- if deck.topics -%} {%- for t in deck.topics -%} {%- unless all_topics contains t -%} {%- assign all_topics = all_topics | push: t -%} {%- endunless -%} {%- endfor -%} {%- endif -%} {%- endfor -%} {%- assign sorted_topics = all_topics | sort -%} {%- for t in sorted_topics -%} {%- endfor -%}
