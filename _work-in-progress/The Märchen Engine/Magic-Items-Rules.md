## Design Anchors

The system below treats magic items as **relationships, not loot math**. That fits the Märchen Engine’s stated table goals: fun, meaningful challenge, character motivation, visible consequences, player engagement, spotlight sharing, and keeping the game moving. ([Ted Tschopp's Homepage][1])

It also uses concepts already present in your rules: the character sheet already distinguishes **Equipment → Operational Gear → Specialized Tools → Elite Gear → Named Items**, and it asks whether customized tools have become famous enough to have names. ([Ted Tschopp's Homepage][2]) The system ties item use to the FAME structure, Characteristics, and Knowledge Areas/Skills, including Fitness, Academic, Mystical, and Emotional approaches. ([Ted Tschopp's Homepage][2])

Mechanically, it uses your existing task-resolution language: skill total, circumstantial difficulty modifiers, 4d8 results, and the existing ladder from failure through Serendipitous success. ([Ted Tschopp's Homepage][3]) The Earthdawn inspiration is specifically the idea of magic items with rank-by-rank effects, hidden lore, required deeds, and powers that unfold as the bearer deepens the bond. FASA’s own Earthdawn example shows Thread Ranks with Deeds, Key Knowledge, and increasing Effects. ([FASA Games][4])

# Story-Threaded Named Items

A **Story-Threaded Named Item** is an object whose magic awakens as its bearer learns its story, performs deeds that harmonize with that story, and lets the item become part of the character’s identity.

A sword does not become great because it is “+2.” It becomes great because it was carried through the ash gate, because the player named it, because the table remembers what happened when it broke, and because the next time it is drawn everyone knows something meaningful is about to happen.

## The Core Rule

A magic item has a **Bond Rank** from 0 to 8.

Rank 0 is a mundane item, a dormant relic, or an item whose magic has not yet accepted the bearer.

Ranks 1–8 are **Story Thread Ranks**. Each rank has:

| Element      | Purpose                                                                   |
| ------------ | ------------------------------------------------------------------------- |
| **Key Lore** | Something the bearer must learn, discover, remember, research, or define. |
| **Deed**     | Something the bearer must do in play, preferably under pressure.          |
| **Gift**     | The power unlocked at that rank.                                          |
| **Sign**     | A sensory or narrative tell that makes the item feel alive.               |
| **Shadow**   | The item’s temptation, flaw, curse, old grief, or dangerous agenda.       |

The bearer only gains a rank after completing the Key Lore, performing the Deed, and succeeding at a **Bond Check**.

# The Design Rubric

Use this rubric whenever you design a magic item, artifact, relic, or legendary tool.

Scores are from 0 to 5.

| Criterion                     | Weight | What “5” Looks Like                                                                                                                       |
| ----------------------------- | -----: | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **Table Fun and Flow**        |    20% | The item creates exciting choices without slowing play.                                                                                   |
| **Character-Gear Engagement** |    20% | The player cares about the item’s name, history, behavior, and future.                                                                    |
| **Discovery and Agency**      |    15% | The player can pursue clues, make choices, and shape the item’s legend.                                                                   |
| **Bounded Power and Balance** |    15% | The item improves play without replacing skills, companions, or challenges.                                                               |
| **Spotlight Sharing**         |    10% | The item creates reasons to involve other PCs, NPCs, Patrons, Sanctuaries, or the Company.                                                |
| **Referee Usability**         |    10% | The item is easy to prep, reveal, advance, and improvise.                                                                                 |
| **Märchen Compatibility**     |    10% | The item uses existing skills, difficulty modifiers, degrees of success, gear categories, time scales, and character identity structures. |

## Target Score

A good item should score **80+**.
A campaign-defining item should score **90+**.
If an item scores below 70, it is probably just a stat bonus with a name.

The main weakness is that a high-rank artifact can become prep-heavy. The mitigation is simple: the Referee only needs to prepare the **current rank and the next rank**. Later ranks can remain suggestive until the campaign earns them.

# Named Item Sheet

Use this for every magic item.

| Field               | Notes                                                                                       |
| ------------------- | ------------------------------------------------------------------------------------------- |
| **Name**            | The item’s known name. It may have deeper names.                                            |
| **Base Item**       | Sword, bow, axe, cloak, book, ship, ring, workshop, idol, mask, saddle, etc.                |
| **Gear Type**       | Equipment, Operational Gear, Specialized Tool, Elite Gear, Named Item.                      |
| **Source Aspect**   | Fitness, Academic, Mystical, Emotional, or a blend.                                         |
| **Signature Skill** | The skill most often used to bond with or awaken the item.                                  |
| **Virtue**          | What the item rewards. Examples: courage, mercy, precision, loyalty, wonder, rebellion.     |
| **Shadow**          | What the item tempts. Examples: pride, vengeance, cowardice, obsession, cruelty, isolation. |
| **Bond Rank**       | Current rank with this bearer.                                                              |
| **Rank Ceiling**    | Maximum rank: 2, 4, 6, or 8.                                                                |
| **Echo**            | A small pool earned through story engagement.                                               |
| **Dissonance**      | Trouble caused by misusing or resisting the item’s story.                                   |
| **Known Lore**      | What the bearer knows so far.                                                               |
| **Hidden Lore**     | What the Referee knows so far.                                                              |
| **Current Quest**   | The next Key Lore and Deed.                                                                 |

# Rank Ceiling

Not every magic item should go to Rank 8.

| Rank Ceiling | Item Type                                | Campaign Role                                                |
| -----------: | ---------------------------------------- | ------------------------------------------------------------ |
|        **2** | Charm, heirloom, lucky tool, minor relic | Useful and flavorful.                                        |
|        **4** | Major magic item                         | A signature item for a long campaign arc.                    |
|        **6** | Legendary item                           | A defining part of a hero’s identity.                        |
|        **8** | Mythic artifact                          | A thing that can reshape communities, nations, or cosmology. |

Most player-facing magic items should top out at **Rank 4**. A party may have one or two Rank 6 items. Rank 8 items should be rare enough that the whole table knows they matter.

# Bond Slots

A Hero may have **three active Story Threads**:

1. **Hand** — a weapon, tool, instrument, implement, or active object.
2. **Ward** — armor, shield, cloak, ring, amulet, tattoo, mask, or protective object.
3. **Heart** — a token, heirloom, book, banner, relic, holy symbol, keepsake, or social/spiritual object.

A Hero may carry more Named Items, but only active Story-Threaded items grant Gifts.

A matched pair, such as twin blades, paired pistols, bracers, or wedding rings, may occupy one slot if they share one item sheet and one Bond Rank.

A Company relic, Sanctuary relic, ship, banner, forge, table, shrine, or spellbook may be bonded to the **Company** instead of one Hero. Company items should grant more group effects and fewer personal combat bonuses.

# The Bond Check

When a Hero tries to unlock the next rank, use normal Märchen task resolution.

The player describes the approach. The Referee chooses the most fitting Aspect, Characteristic, and Skill.

Examples:

| Approach                              | Possible Skill                              |
| ------------------------------------- | ------------------------------------------- |
| Researching a relic’s history         | Investigation, Traditions, Social Sciences  |
| Reforging, repairing, or modifying it | Engineering, Physical Sciences, Order       |
| Awakening a spell-like property       | Arcane, Metaphysics, Theurgy, Shamanism     |
| Bargaining with the spirit inside     | Persuasion, Etiquette, Leadership, Animism  |
| Proving worth through use             | Melee, Ranged Combat, Athletics, Ruggedness |
| Performing its legend publicly        | Performance, Style, Leadership              |
| Understanding its curse               | Insight, Gnosis, Order, Traditions          |

## Bond Difficulty by Rank

Use the rank being unlocked.

| New Rank | Difficulty Modifier | Difficulty Name   |
| -------: | ------------------: | ----------------- |
|        1 |                ⚅ +4 | Routine           |
|        2 |                ⚅ +0 | Average           |
|        3 |                ⚅ -4 | Difficult         |
|        4 |                ⚅ -5 | Very Difficult    |
|        5 |               ⚅ -10 | Formidable        |
|        6 |               ⚅ -15 | Heroic            |
|        7 |               ⚅ -25 | Nearly Impossible |
|        8 |               ⚅ -30 | Godly             |

Ranks 5–8 are not meant to be unlocked by a lone roll in a quiet room. They should require allies, Patrons, Sanctuaries, dangerous journeys, rare components, public vows, or life-changing consequences.

## Bond Check Outcomes

| Result               | Bond Outcome                                                                                                                                                                 |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Catastrophic**     | The item rejects the bearer for now. Mark 2 Dissonance and introduce a serious complication.                                                                                 |
| **Costly**           | The rank does not unlock. The player learns what price must be paid next. Mark 1 Dissonance.                                                                                 |
| **Fail**             | The rank does not unlock, but nothing is lost. The Referee gives a new lead.                                                                                                 |
| **Minor Fail**       | The rank does not unlock yet, but the player may keep one piece of progress: Key Lore, Deed, or material component.                                                          |
| **Fail in Style**    | The rank does not unlock, but the story advances. Reveal a clue, rival, cost, hidden flaw, or alternate deed.                                                                |
| **Tie**              | The rank unlocks, but with a visible Sign or minor obligation.                                                                                                               |
| **Succeed**          | The rank unlocks normally.                                                                                                                                                   |
| **Succeed in Style** | The rank unlocks, and the player may define one harmless detail about the item’s history or appearance.                                                                      |
| **Critical**         | The rank unlocks, and the bearer gains 1 Echo. The next Key Lore is also hinted.                                                                                             |
| **Serendipitous**    | The rank unlocks, and something unrelated but favorable is revealed: a lost ally, a hidden map, a forgotten Sanctuary, a dormant Company benefit, or a truth about an enemy. |

Failure should not dead-end the item. It should create a better adventure.

# Echo

**Echo** is the item answering the bearer.

A Hero may hold Echo up to the item’s Bond Rank, maximum 3. Rank 7–8 artifacts may hold up to 4.

## Earning Echo

A Hero earns 1 Echo when one of these happens:

| Trigger                                             | Example                                                                        |
| --------------------------------------------------- | ------------------------------------------------------------------------------ |
| The item changes the player’s decision              | “I was going to flee, but Hearthbrand will not abandon someone behind a door.” |
| The player brings the item’s history into the scene | “This was forged from a broken temple bell; I ring it against the stone.”      |
| The item helps spotlight another character          | “I hand the spear to my brother-in-arms and trust him with its power.”         |
| The bearer accepts a meaningful cost                | “I will lose the duel if it means keeping the oath.”                           |
| The table names a memorable item moment             | The whole table remembers the scene and agrees it mattered.                    |

A player should normally earn no more than **1 Echo per item per session**.

## Spending Echo

Spend 1 Echo to do one of the following:

| Spend                 | Effect                                                                                                 |
| --------------------- | ------------------------------------------------------------------------------------------------------ |
| **Awakened Edge**     | Add ⚅ +1 to a roll directly tied to the item’s Virtue or Gift.                                         |
| **Remembered Use**    | Ask the Referee one question about the item’s past, maker, former bearer, or intended purpose.         |
| **Resonant Sign**     | Make the item produce an omen: warmth, ringing, shadow, scent, pulse, whispered name, reflected image. |
| **Steady the Thread** | Reduce Dissonance by 1 after the bearer takes a meaningful step toward reconciliation.                 |
| **Activate a Gift**   | Use a Gift that requires Echo.                                                                         |

Echo is not a generic hero point. It only works when the item matters.

# Dissonance

**Dissonance** is the item and bearer falling out of harmony.

Mark 1 Dissonance when:

| Trigger                                     | Example                                              |
| ------------------------------------------- | ---------------------------------------------------- |
| The bearer acts against the item’s Virtue   | Using a mercy relic for cruelty.                     |
| The bearer gives in to the item’s Shadow    | Letting a vengeance blade choose the target.         |
| The bearer breaks a vow made during bonding | Swearing to protect a Sanctuary, then abandoning it. |
| A Bond Check goes badly                     | Costly or Catastrophic result.                       |
| The player accepts a Shadow Bargain         | The Referee offers power with a consequence.         |

At **3 Dissonance**, the item becomes troubled. Choose one:

1. One Gift becomes unavailable until the bearer makes amends.
2. The next use of a Gift creates a complication.
3. The item attracts an enemy, rival, spirit, claimant, or curse.
4. The item’s Shadow appears in the world as a problem to solve.

Dissonance should complicate play, not punish the player for caring.

# Shadow Bargains

Once per session, the Referee may offer a Shadow Bargain:

“You can gain ⚅ +2 or 1 Echo now, but the item’s Shadow gets a say.”

The player must know the likely kind of consequence before accepting. No hidden gotchas.

Examples:

| Item Shadow | Bargain Consequence                                               |
| ----------- | ----------------------------------------------------------------- |
| Vengeance   | You win the fight, but someone begs you to stop and you do not.   |
| Pride       | Your victory humiliates an ally or Patron.                        |
| Obsession   | You ignore another clue to chase the item’s mystery.              |
| Cowardice   | The item protects you, but someone else is exposed.               |
| Hunger      | The item consumes a memory, favor, asset, spell, or relationship. |

# Rank Structure

Each rank should feel different. Do not make every rank “more damage.”

| Rank | Name          | What It Should Add                                                           |
| ---: | ------------- | ---------------------------------------------------------------------------- |
|    0 | Dormant       | The item is mundane, sleeping, unknown, or unwilling.                        |
|    1 | Named         | The item has a name, Sign, and first minor Gift.                             |
|    2 | Proven        | The item becomes genuinely useful in its core role.                          |
|    3 | Remembered    | The item reveals a past bearer, deed, wound, or purpose.                     |
|    4 | Renowned      | The item becomes a signature part of the character.                          |
|    5 | Entwined      | The item affects allies, Company, Sanctuary, Patron, or community.           |
|    6 | Legendary     | The item can change the outcome of major scenes.                             |
|    7 | Mythic        | The item can make impossible deeds merely heroic under the right conditions. |
|    8 | World-Storied | The item changes the campaign world, not just the bearer.                    |

# Power Budget

Use these limits to keep items exciting but bounded.

| Rank | Mechanical Ceiling                                                                                       |
| ---: | -------------------------------------------------------------------------------------------------------- |
|    1 | No raw combat bonus, or ⚅ +1 only in a narrow identity-defining situation.                               |
|    2 | ⚅ +1 to one narrow use, one minor property, or one utility trick.                                        |
|    3 | Once per session active effect, minor defensive twist, memory power, or situational ⚅ +1 broadening.     |
|    4 | ⚅ +2 to a narrow use, remove one drawback in one situation, or add one significant property.             |
|    5 | Group, Sanctuary, Company, Patron, Contact, or ally-facing power. Avoid increasing personal damage here. |
|    6 | Once per session or once per Season legendary effect with a cost.                                        |
|    7 | ⚅ +3 narrow maximum, or reduce a Nearly Impossible task to Heroic when the item’s full story applies.    |
|    8 | World-changing story authority. No further numeric bonus.                                                |

## Stacking Rule

Item bonuses do not stack with other item bonuses. Use the highest applicable item bonus.

A magic item should never grant more than **⚅ +3** to a roll.

The best Gifts are not bigger numbers. They are new choices, altered consequences, opened doors, saved allies, revealed secrets, and story authority.

# Gift Families

When designing a Gift, choose one of these.

## 1. Edge Gifts

A small modifier in a specific situation.

Examples:

| Gift                  | Effect                                                          |
| --------------------- | --------------------------------------------------------------- |
| **Defender’s Edge**   | ⚅ +1 when protecting someone named aloud.                       |
| **Duelist’s Edge**    | ⚅ +1 in a formal duel where both sides acknowledge the contest. |
| **Craftsman’s Edge**  | ⚅ +1 when repairing or improving items of the same type.        |
| **Pilgrim’s Edge**    | ⚅ +1 when navigating toward a known Sanctuary.                  |
| **Oathkeeper’s Edge** | ⚅ +1 when acting to fulfill a spoken promise.                   |

## 2. Property Gifts

The item gains, modifies, suppresses, or conditionally ignores a gear property.

Examples:

| Gift                       | Effect                                                                               |
| -------------------------- | ------------------------------------------------------------------------------------ |
| **Light in Trusted Hands** | A normally awkward item counts as Light for its bonded bearer only.                  |
| **Remembered Balance**     | Ignore one penalty from Heavy, Bulky, or Unbalancing when the item’s Virtue applies. |
| **Returning**              | A thrown weapon returns at the end of the Once if it missed and has a clear path.    |
| **Patient Edge**           | A weapon gains Defensive (1) if the bearer did not attack this round.                |
| **Old Hook**               | A weapon gains Wrap-Around when used against shields, gates, ropes, or bindings.     |

## 3. Memory Gifts

The item remembers.

Examples:

| Gift                     | Effect                                                                                    |
| ------------------------ | ----------------------------------------------------------------------------------------- |
| **Whisper of the Maker** | Spend Echo to ask what kind of hand made, repaired, or altered an object.                 |
| **Blood in the Grain**   | The item reveals whether it has been used in betrayal, mercy, murder, oath, or sacrifice. |
| **Map of Use**           | Once per session, the item points toward a place where it was once used.                  |
| **Witness**              | The item can confirm whether a spoken account matches something it has seen.              |
| **Ancestral Habit**      | The bearer may briefly perform a mundane task as a former bearer would have done it.      |

## 4. Ward Gifts

The item changes harm, consequence, or exposure.

Examples:

| Gift                     | Effect                                                                                               |
| ------------------------ | ---------------------------------------------------------------------------------------------------- |
| **Turn the Worst Aside** | Once per session, reduce a harm-related result by one degree if the bearer interposes the item.      |
| **Hold the Line**        | Spend Echo to prevent forced movement while defending someone behind you.                            |
| **Shared Shelter**       | An adjacent ally receives the item’s minor protection if the bearer gives up their own next attack.  |
| **Last Strap Holds**     | The item breaks a strap, clasp, scabbard, or ornament instead of taking worse damage.                |
| **Mercy of the Relic**   | On a Critical success, the bearer may choose capture, disarm, or revelation instead of greater harm. |

## 5. Channel Gifts

The item expresses a force, spirit, element, style, or magical practice.

Examples:

| Gift                    | Effect                                                                           |
| ----------------------- | -------------------------------------------------------------------------------- |
| **Flame Remembered**    | Change the damage expression or fictional effect to fire for one attack or task. |
| **River-Hand**          | Spend Echo to move a small object, weapon, or tool within Close range.           |
| **Bell of Order**       | Suppress a minor chaotic magical effect for a Moment.                            |
| **Moon-Mirror**         | Reflect an illusion, disguise, or hidden emotion as an omen.                     |
| **Thunder in the Haft** | On Succeed in Style, knock open an unsecured door, shield, or formation.         |

## 6. Company Gifts

The item strengthens the group.

Examples:

| Gift                     | Effect                                                                                    |
| ------------------------ | ----------------------------------------------------------------------------------------- |
| **Company Standard**     | Once per session, an ally who invokes the item’s story gains ⚅ +1.                        |
| **Shared Name**          | Another PC may contribute to the item’s Bond Check with a relevant skill.                 |
| **Sanctuary Key**        | The item reveals, opens, protects, or improves one Sanctuary.                             |
| **Patron’s Mark**        | The item creates or strengthens a relationship with a Patron, Ally, or Contact.           |
| **Remember Us Together** | When the item earns Echo through another PC’s spotlight, both characters mark the moment. |

## 7. Craft Gifts

The item improves downtime, creation, repair, upkeep, or transformation.

Examples:

| Gift                       | Effect                                                                                          |
| -------------------------- | ----------------------------------------------------------------------------------------------- |
| **Never Dulls**            | The item does not need ordinary maintenance unless Dissonant.                                   |
| **Good Pattern**           | Reduce time or cost for a related crafting task using the existing degree/time tools.           |
| **Teaches the Hand**       | The bearer may train a relevant skill during downtime if they use the item as tutor or example. |
| **Takes the New Shape**    | At Rank 4+, the item may be reforged without losing Bond Rank.                                  |
| **Hungry for Better Work** | The item demands rare materials but improves when incorporated.                                 |

# How Items Become Named

There are three ways.

## 1. Found Relics

The item already has a hidden story. The Referee prepares at least Rank 1 and Rank 2.

Use this for ancient blades, lost rings, saintly bones, enchanted books, cursed armor, forgotten masks, heirlooms, and strange tools.

## 2. Forged Relics

The item is created by a character, artisan, Patron, spirit, or Sanctuary.

The player and Referee define the Virtue and Shadow together. The first Key Lore is not “what was this?” but “what did we intend this to become?”

Use this for player-crafted weapons, alchemical devices, wizard staves, holy symbols, tools, ships, golems, spellbooks, and banners.

## 3. Emergent Relics

A mundane item becomes Named because play made it memorable.

At the end of a session, the table may give an item a **Legend Mark** if it was central to a memorable moment.

An item becomes Rank 1 when it has three Legend Marks and the table gives it a name.

Good Legend Mark triggers:

| Trigger                | Example                                                 |
| ---------------------- | ------------------------------------------------------- |
| Saved a life           | The rope that held when everyone thought it would snap. |
| Failed memorably       | The pistol that jammed at the worst possible moment.    |
| Became funny or iconic | The pan that keeps being used as a shield.              |
| Took a cost            | The cloak burned while hiding refugees.                 |
| Was named aloud        | “This is no shovel. This is Gravefriend.”               |
| Changed a decision     | The player chose the hard road because of the item.     |

This is the rule that makes players care about ordinary gear.

# Key Lore

Key Lore should be specific, actionable, and tied to play.

Good Key Lore examples:

| Rank | Key Lore Type                                                                          |
| ---: | -------------------------------------------------------------------------------------- |
|    1 | The item’s first name, current name, or nickname.                                      |
|    2 | The maker, material, homeland, school, tradition, or first owner.                      |
|    3 | The item’s first great deed.                                                           |
|    4 | The item’s greatest failure, shame, curse, or betrayal.                                |
|    5 | The item’s relationship to a living community, bloodline, Sanctuary, Patron, or enemy. |
|    6 | The price paid to create, preserve, steal, or awaken it.                               |
|    7 | The item’s true purpose, not merely its function.                                      |
|    8 | The new legend the bearer will add to the world.                                       |

Do not hide Key Lore behind a single roll. Every failed attempt should create another path to the truth.

# Deeds

A Deed should be something the player chooses to do, not a chore.

Good Deeds:

| Deed Type              | Example                                                                    |
| ---------------------- | -------------------------------------------------------------------------- |
| **Echo Deed**          | Repeat the item’s first great deed in a new context.                       |
| **Correction Deed**    | Repair or redeem the item’s greatest failure.                              |
| **Contradiction Deed** | Refuse the Shadow when it would be easy to indulge it.                     |
| **Witness Deed**       | Publicly declare, display, perform, or defend the item’s story.            |
| **Craft Deed**         | Repair, reforge, repaint, consecrate, sharpen, rebuild, or alter the item. |
| **Kinship Deed**       | Involve another PC, an Ally, a Patron, a Sanctuary, or a community.        |
| **Sacrifice Deed**     | Give up safety, status, wealth, time, secrecy, advantage, or comfort.      |
| **Inheritance Deed**   | Accept, reject, or transform the legacy of a previous bearer.              |

Bad Deeds:

| Bad Deed                                | Why It Is Weak             |
| --------------------------------------- | -------------------------- |
| “Kill 50 enemies”                       | Encourages grinding.       |
| “Spend 1,000 coins”                     | Makes magic transactional. |
| “Wait a month”                          | Does not create play.      |
| “Find the only clue in the dungeon”     | Can stall the item.        |
| “Do what you were already doing anyway” | Does not change behavior.  |

# Item Design Procedure

Use this seven-step process.

## Step 1: Choose the Base Item

Start mundane.

A magic sword is still a sword. A magic bow still cares about range, ammunition, line of sight, and the bearer’s skill. A magic axe still has weight, edge, haft, maintenance, and consequences.

The base item’s mundane qualities should matter even at high rank.

## Step 2: Choose Source Aspect

| Source Aspect | Item Themes                                                                             |
| ------------- | --------------------------------------------------------------------------------------- |
| **Fitness**   | Weapons, armor, tools, movement, endurance, athletic prowess, bodily mastery.           |
| **Academic**  | Books, maps, instruments, devices, lenses, laboratories, methods, craft, memory.        |
| **Mystical**  | Spell vessels, charms, spirit objects, divine relics, occult patterns, transformations. |
| **Emotional** | Banners, rings, masks, crowns, instruments, heirlooms, oaths, leadership, reputation.   |

An item may have two Source Aspects, but one should dominate.

## Step 3: Choose Virtue and Shadow

The Virtue rewards the behavior you want at the table.

The Shadow tempts the behavior that creates drama.

Examples:

| Virtue    | Shadow                          |
| --------- | ------------------------------- |
| Mercy     | Weakness, hesitation, martyrdom |
| Courage   | Recklessness, pride             |
| Loyalty   | Blind obedience, tribalism      |
| Justice   | Vengeance, cruelty              |
| Wonder    | Obsession, distraction          |
| Precision | Perfectionism, coldness         |
| Freedom   | Betrayal, rootlessness          |
| Order     | Tyranny, rigidity               |
| Truth     | Exposure, humiliation           |
| Beauty    | Vanity, manipulation            |

## Step 4: Choose Rank Ceiling

Use 2, 4, 6, or 8.

Do not write eight ranks for every item. A Rank 2 heirloom can be more beloved than a Rank 8 artifact if it matters in play.

## Step 5: Write Rank 1 and Rank 2

Only the current rank and next rank need detail.

For each rank, write:

1. Key Lore
2. Deed
3. Gift
4. Sign
5. Shadow complication, if any

## Step 6: Leave Higher Ranks as Questions

For future ranks, write prompts instead of answers.

Examples:

| Rank | Prompt                                              |
| ---: | --------------------------------------------------- |
|    3 | Who first used this item heroically?                |
|    4 | What terrible thing was done with it?               |
|    5 | Who else has a claim to it?                         |
|    6 | What price does the item still demand?              |
|    7 | What was it truly made to accomplish?               |
|    8 | What must change in the world for its story to end? |

This lets the item grow with the campaign.

## Step 7: Score the Item

Use the rubric. If the item scores low on engagement, add Key Lore. If it scores low on balance, replace a bonus with a choice. If it scores low on spotlight, add a Company Gift or Kinship Deed.

# Example Item: Hearthbrand, the Door-Axe

**Base Item:** One-handed axe, usable as weapon and tool
**Source Aspect:** Fitness + Emotional
**Signature Skills:** Melee, Athletics, Leadership, Traditions
**Virtue:** No one is abandoned
**Shadow:** Every problem becomes a siege
**Rank Ceiling:** 6
**Sign:** The axe smells faintly of smoke and warm iron when a threshold is threatened.

## Rank 1 — The Name at the Door

| Element      | Detail                                                                                                                   |
| ------------ | ------------------------------------------------------------------------------------------------------------------------ |
| **Key Lore** | Learn or give the axe its name: Hearthbrand.                                                                             |
| **Deed**     | Stand in a doorway, breach, gate, bridge, or passage to protect someone behind you.                                      |
| **Gift**     | When physically interposing yourself to protect another, gain ⚅ +1 to the relevant Melee, Athletics, or Ruggedness roll. |
| **Sign**     | The haft grows warm where the hand grips it.                                                                             |
| **Shadow**   | The bearer feels reluctant to retreat, even when retreat is wise.                                                        |

## Rank 2 — The Hinge-Forged Blade

| Element      | Detail                                                                                                                                                      |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |

# Referee Guidance

## Give Every Item Three Ways Forward

For each rank, prepare at least three possible routes:

1. A person who knows something.
2. A place that reveals something.
3. A deed that can awaken something even without full knowledge.

This prevents the item from stalling.

## Let Players Add Truth

On Succeed in Style, Critical, or Serendipitous results, let the player define one detail:

| Player May Define          | Referee May Add                     |
| -------------------------- | ----------------------------------- |
| A former bearer’s nickname | Why that bearer is controversial    |
| A sensory sign             | What creature also notices it       |
| A maker’s technique        | What rare material is still missing |
| A heroic deed              | The cost history forgot             |
| A place tied to the item   | Who currently controls it           |

This makes the item belong to the table.

## Keep Powers Concrete

A weak Gift says: “The sword is magical.”

A strong Gift says: “When you defend someone whose name you spoke aloud, the blade flashes with hearth-light and grants ⚅ +1.”

Concrete powers are easier to remember and easier to love.

## Use Items to Create Adventure

Each rank should point toward one of these:

| Hook Type   | Example                                                                             |
| ----------- | ----------------------------------------------------------------------------------- |
| **Person**  | A maker, heir, rival claimant, former bearer, spirit, scholar, thief, Patron.       |
| **Place**   | Ruined forge, Sanctuary, grave, battlefield, hidden library, sacred tree, old ship. |
| **Faction** | Guild, church, bloodline, Company, school, order, cult, village, enemy house.       |
| **Choice**  | Preserve the old purpose, redeem it, break it, or transform it.                     |
| **Cost**    | Lose secrecy, status, coin, safety, time, comfort, innocence, or certainty.         |

# Optional Rule: Company Relics

A Company may bond one shared item: banner, map, ship, campfire stone, patron’s seal, sanctuary key, spell table, ancestral idol, relic wagon, or oath-book.

A Company Relic has a Bond Rank equal to the number of major Company Deeds completed, maximum 4 unless the campaign is explicitly about that relic.

Company Relic Gifts should do things like:

| Rank | Gift Example                                                                                          |
| ---: | ----------------------------------------------------------------------------------------------------- |
|    1 | The Company can always find its way back to its first Sanctuary.                                      |
|    2 | Once per session, a PC may spend Echo from the relic to aid another PC.                               |
|    3 | The relic reveals when a Patron, Ally, Contact, Rival, or Enemy is relevant to the current situation. |
|    4 | Once per Season, the Company may call on the relic to protect, restore, or reveal a Sanctuary.        |

# Optional Rule: Cursed Named Items

A cursed item is not “bad loot.” It is a Named Item whose Shadow is awake before its Virtue is understood.

Cursed item rules:

1. The first Gift is stronger than normal.
2. The first Key Lore is hidden or false.
3. The item starts with 1 Dissonance.
4. The Referee must telegraph the danger.
5. The player must always have a path to cleanse, redeem, master, abandon, or transform it.

A curse should create decisions, not remove control.

# What This System Optimizes

This system is meant to make players say things like:

“I bring my grandfather’s hammer.”

“I do not draw the sword yet. It has not forgiven me.”

“I know the ring wants me to lie, but I need its help.”

“Can my shield remember the last person it failed to protect?”

“We should return to the Sanctuary. The banner has been pulling toward it all session.”

That is the sweet spot: the item is mechanically useful, but its real power is that it makes the character more vivid, the table more engaged, and the campaign more memorable.
